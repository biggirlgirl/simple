关于JavaScript 

W3School中是这样描述的：
JavaScript 是世界上最流行的编程语言。
这门语言可用于 HTML 和 web，更可广泛用于服务器、PC、
笔记本电脑、平板电脑和智能手机等设备。

JavaScript 是脚本语言，一种轻量级的编程语言。

HTML+CSS只能完成静态的页面，也可以加一些动画效果，
而使用 JavaScript 则可以增加网页的交互功能，比如：点击按钮登录注册等

在这里要分清 JavaScript 和 Java 的关系
JavaScript 和 Java 是两种不同的语言。
JavaScript 和 Java 的关系就好比 雷锋 和 雷峰塔 的关系。


JavaScript 注意事项：
1.区分字母的大小写，name和Name是两个不同的标识符
2.JS代码忽略空格和换行，一行代码可以分成多行书写
3.分号作为一个语句结束的标志，JS代码可以不写分号，但是建议大家写上
4.代码的注释：单行注释和多行注释，被注释的代码不会执行
	// 双斜杠表示单行注释
	/* 注释的内容 */ 多行注释


使用JS的方式：
方法1：
	在html中直接使用 <script>...</script> 标签编写JS代码
方法2：
	使用 <script src="file.js"></script> 导入外部JS文件



JavaScript 是面向对象的编程语言

对象：就是实实在在存在的一个实体，该实体在DOM中其实
就是一个节点对象。拥有 属性 及 方法。

属性：属性就是一个对象所拥有的数据（数据包括数据的类型和数据的值）

方法：方法就是对象所具有的功能。通过对象的功能使用对象的属性来实现一些操作。

例如：人类中的一个对象张三，每一个对象都有自己的属性和方法
张三  属性：age：18； height：180；weight：80；
     方法：work：WEB前段工程师；study：在智游学习HTML5


脚本语言是一种【解释性】的语言

解释性语言
代码是不需要要编译 直接可以执行
代码按行执行，前边的代码不会受到后边的代码错误的影响

编译性语言
先把代码编译为可执行文件 比如exe文件，然后才能执行。
假如语法有错误，或是代码在编译的时候出错了，就没办法运行


JS是一种【弱类型】的语言

弱类型
变量本身没有类型，但是变量的值有类型。即变量的类型是由变量的值决定的。
var a = 3;     表示整数
var b = ‘123’; 表示字符串

强类型
变量的类型是在变量在声明的时候决定的。
int a  = 3;        表示整数
String b = ‘你好’;  表示字符串
